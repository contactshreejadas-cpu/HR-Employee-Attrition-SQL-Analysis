# HR-Employee-Attrition-SQL-Analysis
HR Attrition Analysis using SQL

## Objective
Analyze employee attrition and identify key drivers impacting retention.

## Dataset
IBM HR Analytics Employee Attrition Dataset (1,470 records)

## Tools
- SQLite
- DB Browser for SQLite
- SQL

## Analysis Performed
- Overall attrition rate calculation
- Department-wise attrition analysis
- Salary band vs attrition
- Experience and age analysis
  
## Key Insights
- Overall attrition rate is approximately **16%**
- The **Sales department** exhibits the highest attrition risk compared to other departments
- Employees in **lower salary bands** show significantly higher attrition rates
- **Early-career employees** (with fewer years at the company) are more likely to leave
- **Younger age groups** demonstrate higher attrition, indicating challenges in early-stage employee retention

## Conclusion
This project provides data-driven HR insights that can support
employee retention and strategic workforce planning.
